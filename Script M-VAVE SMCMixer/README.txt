Move the M-VAVE SMCMixer folder along the path C:\Users\Username\Documents\Image-Line\FL Studio\Settings\Hardware
Then run the Mini suite utility downloaded from the manufacturer's website http://www.cuvave.com/appdownload
And click the import config button and select the SMC Mixer config CC Mode file
Select CC Mode by pressing shift + ➔
Everything is ready
The arrows ← → Change the mixer tracks from 9-16 to 17-24 and so on
Square ° Turns off the effects on the mixer track
The letter m Turns off the sound of the mixer track
The letter s Turns off all mixer tracks except the selected one
The letter r enables recording of the selected mixer track
